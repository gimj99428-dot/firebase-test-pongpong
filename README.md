# ✈️ 여행자 커뮤니티 및 내비게이션 웹페이지

## 🌟 프로젝트 개요

이 웹페이지는 **React.js**를 기반으로 구축된 여행 공유 플랫폼입니다. **Firebase Authentication**을 통한 안전한 사용자 인증과 **Cloud Firestore**를 활용한 실시간 데이터 공유 기능을 제공하여, 사용자들이 여행 정보를 효율적으로 탐색하고 자신의 경험을 공유할 수 있도록 설계되었습니다.

### 기술 스택

| 구분 | 사용 기술 | 목적 |
| :--- | :--- | :--- |
| **Frontend** | **React.js** (JavaScript) | UI 구축 및 `useState`를 활용한 상태 기반 라우팅 |
| **Authentication** | **Firebase Authentication** | 회원가입, 로그인, 로그아웃 및 사용자 세션 관리 |
| **Database** | **Cloud Firestore** | 여행 게시글의 실시간 저장, 조회 및 보안 관리 |

---

## 💡 핵심 기능 (Core Features)

| 기능 영역 | 설명 |
| :--- | :--- |
| **로그인 / 회원가입** 🔑 | 이메일 기반의 사용자 인증을 제공합니다. 로그인 상태는 앱 전체에서 유지되며, 비로그인 시 모든 주요 기능에 대한 접근이 차단됩니다. |
| **로그아웃** 🚪 | 메인 메뉴에서 Firebase 세션을 안전하게 종료하고, **초기 화면 (`/`)**으로 자동 복귀합니다. |
| **여행지 탐색** 🗺️ | **가까운 여행지**와 **먼 여행지**로 구분된 메뉴를 통해 사용자가 원하는 지역 정보로 쉽게 이동할 수 있습니다. |
| **여행 정보 공유** 📝 | 날짜, 이미지 URL, 후기를 작성하여 **Firestore 데이터베이스에 실시간으로 저장**하고 공유합니다. (작성자 ID 자동 기록) |
| **여행 리뷰 조회** ✨ | 다른 사용자들이 공유한 모든 게시물을 **최신순**으로 조회합니다. |
| **조건부 삭제** 🗑️ | **본인이 작성한 게시글에만** 삭제 버튼이 표시되어 개인 데이터를 안전하게 관리할 수 있습니다. |
